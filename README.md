## ðŸš€ Overview

**Theme Creator App** is an interactive web tool built with **Next.js** and **React**, designed for generating custom color palettes.  
Users can add new colors, edit existing ones, rename color roles, copy HEX values, adjust contrast text colors, and delete colors.  
The UI updates in real-time and works fully responsive.

This project was created during the **Neue Fische Web Development Bootcamp** as practice in state management, UI/UX logic, and building practical developer tools.
